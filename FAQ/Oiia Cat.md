# Oiia Cat 使用指南

## 概述
Oiia Cat 是一个有趣的彩蛋功能，让您可以将用户名更改为特殊的 oiia 格式。这个功能结合了视觉和交互元素，为用户提供独特的个性化体验。

## 快速开始

### 基本使用方法
想要使用 Oiia Cat 功能，您只需要简单的两步操作：

1. **准备阶段**：确保您的账户中有"令名牌"和"铁砧"道具
2. **执行操作**：在道具栏中选择这两个道具，然后将您的用户名更改为 "oiia"

就是这么简单！完成操作后，您就能体验到 Oiia Cat 带来的独特效果。
