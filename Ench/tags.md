## 🔧 功能性开关类
> 以下功能可由玩家自行随意修改

| 标签名称 | 含义 | 备注 |
|------|------|---------|
| qys:no_sweep | 禁用横扫之刃 |
| qys:no_ShowBiome| 禁用群系提示 | 注意：存在时不显示群系 |
| qys:no_word | 禁用每日一言 | 屏蔽定时广播 |
| qys:no_sitdown | 禁止坐上楼梯 |
| qys:no_flyRollOver| 禁用鞘翅翻滚 |
| qys:no_harvest | 禁用自动补种 |


## 🎭 状态标记类
> 以下功能不可由玩家自行随意修改，但是可以通过使用魔法来获得

| 标签名称 | 含义 | 生命周期 |
|-------|------------|----------|
| qys:in_afk | 玩家处于挂机状态 | 持续到玩家移动/操作后清除 |
| qys:can_speed | 允许冲撞魔法 | 持续到玩家退出 |
| qys:false_creation | 玩家处于假创造模式 | 持续到玩家退出 |
| qys:ride_ender_dragon | 玩家处于骑乘末影龙状态 | 持续到玩家退出 |
| qys:can_speed | 允许冲撞魔法 | 持续到玩家退出 |


## 💾 数据存储类

| 标签名称 | 含义 | 数据格式 |
|-------|----------|-----------|
| qys:beforeSkin_* | 穿皮肤前的皮肤ID |
| qys_data:elytra:* | 已解锁的鞘翅皮肤列表 | JSON数组格式 |
| qys:onEvent_* | 待触发的自定义事件 |
| qys:runcmd_* | 实体命令执行临时ID |

## 🎯 称号相关

| Tag名称 | 含义 |
|-------|------|
| qys.tag:unset | 锁定称号不可更改 |
| tag:xxx | 玩家拥有的称号池（前缀为tag:） |

## 特殊功能标签

| Tag名称 | 所属系统 | 作用 | 作用对象 |
| --------|------|----------|----------|
| op | 管理员标记 | 绕过部分权限检查 | 玩家 |
| isCat  | 皮肤特效 | 穿戴"猫猫"皮肤时触发猫叫声 | 玩家 |
| naa:no_axolotl_damage  | 美西螈保护 | 禁止美西螈攻击鱼类 | 玩家 |
| player  | 新手标记 | 标识已完成新手教程的玩家 | 玩家 |
| structure | 结构管理系统| 给玩家添加后会打开全局结构管理页面 | 玩家 |
| qys.tag:noAchievementText  | 成就系统 | 给玩家添加后获取成就信息将不会全服播报 | 玩家 |
| qys:firework_open | 烟花系统 | 实体Tag，防止重复点燃 | 实体 |
