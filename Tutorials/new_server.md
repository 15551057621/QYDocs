# 🚀 启动你的第一个Minecraft服务器

开服，其实是一件很简单的事情！  
本指南将手把手教你搭建一个属于自己的Minecraft服务器。  

> **📌 注意**  
> 本文档仅适用于 **Minecraft基岩版 (Bedrock Edition)** 服务器搭建，Java版用户请自行搜索教程  
> 适用系统：Windows 10/11 或 Windows Server 2019/2022  

## 📋 准备工作清单

在开始之前，请确保你已准备好：

- [#] 一台运行 Windows 系统的电脑
- [#] 稳定的网络连接（推荐有线网络）
- [#] 至少 **4GB** 的可用内存
- [#] 至少 **20GB** 的可用磁盘空间
- [#] 基本的计算机操作知识

如果以上你都具备，那么可以接下来教程了

## 1️⃣ 配置运行环境

### 系统要求
  
| 硬件项目 | 最低要求 | 推荐配置 |
|-------|----------|----------|
| 操作系统 | Windows 10 / Windows Server 2019 | Windows 11 / Windows Server 2022 |
| 运行内存 | 2GB | 4GB+ |
| 存储空间 | 10GB | 20GB+ |
| CPU配置 | 2.50GHz的cpu | Intel Core i3 3210 | AMD A8 7600 APU 或同等配置 |

### 安装必要组件
  
1. **下载并安装 Microsoft Visual C++ Redistributable**
   - 下载地址：[https://aka.ms/vs/17/release/vc_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)
   - 双击运行安装程序
   - 选择"同意条款"，点击"安装"
   - 安装完成后可能需要重启电脑
   > **💡 提示**：这是服务器运行的基础依赖库，必须安装！

2. **确认系统更新**  
按 Win + R，输入 winver，查看系统版本  
确保系统已安装最新更新

## 2.️ 选择适合的服务端

如果你已完成环境配置，恭喜！现在可以挑选你的服务器核心了。  
以下是目前主流的基岩版服务端对比：  

| 服务端 | 特点 | 适合人群 | 推荐指数 |
|--------|------|----------|----------|
| **LeviLamina** | 基于BDS，插件生态丰富<br>性能优秀，稳定性强<br>社区活跃，文档齐全 | 希望使用插件扩展功能的服主 | ⭐⭐⭐⭐⭐ |
| **BDS (官方)** | Mojang官方出品<br>更新及时，兼容性好<br>原生支持所有特性 | 追求原版体验、不需要插件的服主 | ⭐⭐⭐⭐ |
| **EndStone** | 相对较新的服务端<br>社区生态还在发展中 | 喜欢尝鲜、有Python开发经验的服主 | ⭐⭐ |

**我们的推荐：** 对于新手，建议从 **LeviLamina** 开始，它在保留官方特性的同时提供了强大的插件支持。
  
## 3. 下载服务端程序

### 方案A：使用 LeviLamina（推荐）

   - GitHub Releases: [LeviLamina Releases](https://github.com/LiteLDev/LeviLamina/releases)
